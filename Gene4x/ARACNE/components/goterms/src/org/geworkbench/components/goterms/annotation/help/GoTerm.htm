Gene Ontology

The Gene Ontology component is located in the View Area in the top-right panel of the application. This component provides categorization of genes in terms of function their products perform, their cellular localization or involvement in high level biological processes. The actual category definitions are provided by the Gene Ontology Consortium (http://www.geneontology.org ) which we make use of in generating a tree structure which approximates the Directed Acyclic Graph structure used by the Consortium.

Specifically, this component computes and provides enrichment statistics using the Tree View, Table View or P-value trends and three tabbed regions in the Graphical User Interface.
