P-Value Trends

This plot displays how a P-Value varies when the number of genes (as mapped from Affy Probe IDs) in the selected list varies from zero to the existing size of the list. Every profile depicts an enabled category shown in the Table View. This help section describe the table view Basic Usage.

Basic Usage

The following summarizes the description of the P-Value Trend:

Slider: The slider on the right is a log-scaled slider that specifies a minimum P-Value for every profile to have for it to be shown in the display

Step-size: The ‘Step Size’ widget allows for setting the number of genes in increments to be used for the P-Value plot (the data points on the x-axis to be used). This allows for smoothing of the curve especially for very large gene list sizes.

Plot: The ‘Plot’ will redraw the plot based on selections in the Tree View and widget settings in the ‘P-Value setting’ tab.

Save Profiles: The ‘Save Profile’ allows for the computed profiles to be saved to the file-system as tab-delimited values
